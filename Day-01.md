##Python Operators

## Overview

Today I learned about Python Operators, which are special symbols used to perform operations on variables and values. Operators help us perform calculations, compare values, assign data, and make decisions in programs.

## Topics Covered

### 1. Arithmetic Operators

Arithmetic operators are used for mathematical calculations.

* `+` Addition
* `-` Subtraction
* `*` Multiplication
* `/` Division
* `%` Modulus (remainder)
* `**` Exponentiation (power)
* `//` Floor Division

**Example:**

```python
x = 5
y = 2

print(x + y)   # 7
```

### 2. Bitwise Operators

Bitwise operators work on binary representations of numbers.

* `&` AND
* `|` OR
* `^` XOR
* `~` NOT
* `<<` Left Shift
* `>>` Right Shift

### 3. Assignment Operators

Assignment operators are used to assign and update values in variables.

* `=`
* `+=`
* `-=`
* `*=`
* `/=`
* `%=`
* `**=`
* `//=`

### 4. Comparison Operators

Comparison operators compare two values and return either `True` or `False`.

* `==`
* `!=`
* `>`
* `<`
* `>=`
* `<=`

### 5. Logical Operators

Logical operators are used to combine conditions.

* `and`
* `or`
* `not`

### 6. Identity Operators

Identity operators check whether two variables refer to the same object in memory.

* `is`
* `is not`

### 7. Membership Operators

Membership operators check whether an element exists in a sequence.

* `in`
* `not in`

**Example:**

```python
numbers = [1, 2, 3]

print(2 in numbers)      # True
print(5 not in numbers)  # True
```

## Key Learnings

* Python provides seven major categories of operators.
* Arithmetic operators are useful for calculations.
* Comparison and logical operators are important for decision-making.
* Identity operators check object references, not just values.
* Membership operators help verify whether an item exists in a collection.
